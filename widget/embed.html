Add the Marv AI damage validator to your website in 3 simple steps:

Step 1: Add the CSS

Add this line to your <head> section:

<link rel="stylesheet" href="https://your-static-site.azurestaticapps.net/widget/marv-widget.css">

Step 2: Configure the API

Add this script before the widget script (replace with your Azure Function URL):

<script>
  window.MARV_CONFIG = {
    apiBase: 'https://your-function-app.azurewebsites.net'
  };
</script>

Step 3: Add the Widget Script

Add this line before the closing </body> tag:

<script src="https://your-static-site.azurestaticapps.net/widget/marv-widget.js"></script>

Complete Example

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Website</title>
  
  <!-- Marv Widget CSS -->
  <link rel="stylesheet" href="https://your-static-site.azurestaticapps.net/widget/marv-widget.css">
</head>
<body>
  
  <!-- Your website content -->
  <h1>Welcome to my website</h1>
  
  <!-- Marv Widget Configuration -->
  <script>
    window.MARV_CONFIG = {
      apiBase: 'https://your-function-app.azurewebsites.net'
    };
  </script>
  
  <!-- Marv Widget Script -->
  <script src="https://your-static-site.azurestaticapps.net/widget/marv-widget.js"></script>
</body>
</html>
